# macOS Mojave (dark)
### Theme reproducing the clean and bright look of Apple's OS (but in dark mode)

### Manual installation

Extract the zip file to the themes directory i.e. `/home/USERNAME/.themes`

### Requirements

- GTK+ 3.16 or above
- Murrine and Pixmap theme engines
